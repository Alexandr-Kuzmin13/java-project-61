## Приложение Игры разума
***
Набор из пяти `консольных` математических игр:
* Проверка на чётность
* Калькулятор
* Наибольший общий делитель (НОД)
* Арифметическая прогрессия
* Простое ли число?
***
Движок реализован отдельно, что позволяет дополнять проект новыми играми.
